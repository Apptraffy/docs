# Шаг 1. Установка скрипта.

### 1.1. Получите Traffy Key кампании и скопируйте скрипт ниже.

Если вы создатель рекламной кампании, передайте вашему разработчику **Traffy Key** из приложения **Traffy**. Его можно найти, выбрав карточку кампании на главной странице покупателя и перейдя в **раздел со скриптом**.

Затем скопируйте скрипт и замените значение **resource-id** на ваш **Traffy Key**:

```mdx-code-block
<CodeBlock className="language-html" title="Скрипт Traffy:">
{`<script src="${SCRIPT_SRC}" resource-id="Paste Your Traffy Key" mode="production"></script>`}
</CodeBlock>
```

### 1.2. Разместите скрипт внутрь тега ```<head>``` вашей ```html``` страницы.

Вставьте скопированный скрипт (из шага 1.1) в `<head>` вашего Telegram Miniapp. После этого задания с подпиской станут "с проверкой", что повысит их эффективность, а статус скрипта изменится в карточке кампании.

**Внимание! Скрипт должен быть размещен после скрипта Telegram (https://telegram.org/js/telegram-web-app.js)**

```mdx-code-block
<CodeBlock className="language-html" title="Пример (index.html):">
{`<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script src="https://telegram.org/js/telegram-web-app.js"></script>
    <script src="${SCRIPT_SRC}" resource-id="Paste Your Traffy Key" mode="production"></script>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/index.tsx"></script>
  </body>
</html>`}
</CodeBlock>
```
