# Step 1. Preparation

### 1.1. Obtain the advertising platform script.

If you are the creator of the Seller (advertising platform), send the script to your developer via the Traffy application.
You can copy the script from the advertising platform card.

An example of what the script looks like:
```html
<script src="https://dvq1zz1g273yl.cloudfront.net/index_v1.1.0.min.js" traffy-key="Your Traffy Key" test="true"></script>
```

### 1.2. Place the script inside the ```<head>``` tag of your ```html``` page.

Paste the copied script at the end of the ```<head>``` section in the ```html``` of your Telegram Miniapp. After this, you'll be able to receive and render Traffy tasks.

#### Important! The script should be placed after the Telegram script and before the Miniapp script.

Here is an example of how it might look:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script src="https://telegram.org/js/telegram-web-app.js"></script>
    <script src="https://dvq1zz1g273yl.cloudfront.net/index_v1.1.0.min.js" traffy-key="Traffy Key" test="true"></script>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/index.tsx"></script>
  </body>
</html>
```
