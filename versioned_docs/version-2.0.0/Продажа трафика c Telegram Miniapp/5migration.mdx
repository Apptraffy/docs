# Миграция на версию 2.

### Если вы уже интегрировали ранее скрипт Traffy:

1. Создайте новый ресурс (нового селлера) в новом приложении Traffy, если еще его не создали
2. Замените старый скрипт на новый (пройдите шаг 1)
3. Обратите внимание, что на шаге 2 изменился тип Task. Теперь он выглядит так:
```ts
type Task = {
    id: string; // ранее company_id
    title: string;
    image_url: string | null; // ранее photo_url
    link: string;
}
```
4. На шаге 4 изменилось апи для верификации токена на бэкенде. Теперь с фронтенда нужно отправлять только token,
при этом изменился запрос для верификации токена.
